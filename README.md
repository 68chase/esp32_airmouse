# ESP32 Bluetooth Air Mouse

Forked from project at:  https://github.com/n1rml/esp32_airmouse

### Prerequisites
Install ESP-IDF : https://github.com/espressif/esp-idf

### Installation
Clone the project
```
git clone https://github.com/68chase/esp32_airmouse

cd esp32_airmouse
```
Build
```
idf.py build
```
Flash
```
idf.py flash

```
